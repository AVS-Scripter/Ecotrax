# **App Name**: Ecotrax

## Core Features:

- Environmental Incident Reporting: A form to submit detailed reports about environmental issues, including issue type, description, image upload, and location input with validation and toast notifications.
- Interactive Dashboard Visualization: Displays key environmental metrics and issue trends through interactive charts (e.g., pollution types, reports over time) and summary cards in a clean grid layout.
- Geospatial Issue Mapping: An interactive map view (mock Google Maps integration) to visualize reported incidents by location with clickable markers providing issue type and description in a popup.
- Community Challenges & Leaderboards: Showcases environmental challenges, tracks user progress, and displays a leaderboard with user cards and points to foster community engagement.
- User Authentication UI: Frontend-only user login and registration pages with stylish input fields, validation, and mock social login buttons.
- Dynamic Landing Page: A hero section with a captivating title, environmental awareness subtitle, animated stats counters (reports, areas, users), CTA buttons, smooth scroll navigation, and background animations.
- Responsive Navigation & Theming: A consistent user experience across mobile, tablet, and desktop with a collapsable hamburger menu, active link highlighting, and a dark mode enabled by default with a toggle option.

## Style Guidelines:

- Scheme: Dark by default, embodying a futuristic and eco-conscious mood.
- Primary color: A vibrant neon green (#00FF9F) symbolizing vitality, growth, and digital interface, serving as the main accent and interactive element highlight.
- Background color: A deep, desaturated green-gray (#1A2821) for the main canvases, providing a tranquil yet futuristic base derived from the primary's hue.
- Accent color: A luminous teal (#5AD8A7) for secondary highlights and glowing elements, offering an analogous contrast to the primary green.
- Headline font: 'Space Grotesk' (sans-serif) for titles and hero text, conveying a modern, techy, and scientific aesthetic.
- Body font: 'Inter' (sans-serif) for all general text, ensuring clarity, readability, and a neutral yet professional feel.
- Utilize minimalist outline icons, enhanced with subtle gradient fills or neon glow effects to align with the futuristic and clean green theme.
- Design with mobile-first principles, incorporating responsive grids and stackable components on smaller screens. Elements like cards will feature glassmorphism (blur + transparency) with large rounded corners (xl/2xl), soft shadows, and subtle glow effects.
- A dark gradient (black + deep green) background sets a professional tone, complemented by animated gradient elements for visual depth.
- Implement smooth transitions between pages and elements using Framer Motion. Key animations include fade-in effects on scroll, subtle hover scale effects on buttons, and micro-interactions for an engaging user experience.
- Include dynamic background animations on the landing page, animated stats counters, and loader screens with animated spinners.
